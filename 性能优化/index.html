🔄 while vs forEach 循环性能对比

测试说明

本演示对比了 JavaScript 中两种常用循环方式的性能差异：

while 循环：传统的低级循环，直接操作索引，性能最优

forEach：ES5 引入的高级迭代方法，代码更优雅但有函数调用开销

代码对比

⚡ while 循环

// 传统 while 循环
let i = 0;
while (i < arr.length) {
    // 处理 arr[i]
    sum += arr[i];
    i++;
}

// 特点：
// ✓ 最快的循环方式
// ✓ 无函数调用开销
// ✓ 可随时 break
// ✗ 代码较啰嗦

🔁 forEach

// 高级迭代方法
arr.forEach(function(item) {
    // 处理 item
    sum += item;
});

// 或使用箭头函数
arr.forEach(item => sum += item);

// 特点：
// ✓ 代码简洁优雅
// ✓ 语义清晰
// ✗ 有函数调用开销
// ✗ 不能 break
